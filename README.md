# 🏃 Running application

### What do you need?

- [Flutter](https://flutter.io/get-started/install/)

### 🤘 Let's start

- Clone repo</br>
   - Via https
    ```
    git clone https://github.com/xdk78/schools.git
    ```
   - Via ssh
    ```
    git clone git@github.com:xdk78/schools.git
    ```

- Now enter to app folder
  ```
  cd Schools
  ```
- Install depedencies
  ```
  pub get
  ```
- Running application
  ```
  flutter run
  ```
- Running application in production mode
  ```
  flutter run --release
  ```
